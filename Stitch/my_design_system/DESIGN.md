# Design System Guidelines

This document outlines the core visual principles and token values that define our brand's digital experience.

## Theme Overview

Our design system utilizes a **light** color mode, providing a bright and airy aesthetic.

## Colors

The color palette is built around a "fidelity" variant, aiming for a consistent and trustworthy visual impression.

### Primary Colors
- **Primary Color**: `#2F6FED` - A vibrant blue, used for key interactive elements like buttons, calls to action, and primary navigation.
- **Secondary Color**: `#FF7A1A` - A bold orange, providing a strong accent for secondary actions, highlights, and informative elements.
- **Tertiary Color**: `#EAF2FF` - A very light blue, used for softer accents, backgrounds, or decorative elements that complement the primary palette.
- **Neutral Color**: `#FFF` - Pure white, serving as a clean and bright base for backgrounds, surfaces, and typography.

## Typography

Our typography system is designed for clarity and readability across all content.

- **Headline Font**: `Inter` - Used for all headings and prominent textual elements, ensuring strong visual hierarchy.
- **Body Font**: `Inter` - Applied to all body text, paragraphs, and general content for optimal reading comfort.
- **Label Font**: `Inter` - Utilized for labels, captions, and smaller interface text, maintaining consistency with the overall typographic voice.

## Shape and Form

### Roundedness
Our UI elements feature a **moderate (2)** level of roundedness. This provides a soft, friendly, and contemporary feel without being overly curvilinear.

## Spacing

The design system employs a **normal (2)** level of spacing. This ensures a balanced layout with sufficient whitespace for readability and visual comfort, preventing elements from feeling cramped or excessively spread out.
