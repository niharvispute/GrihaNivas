# Design System Theme

## Overview
This document outlines the core visual theme for our product, ensuring a consistent and harmonious user experience.

## Theme Configuration

### Color Palette
Our color palette is designed for a vibrant and engaging user interface.
-   **Color Mode**: `light`
-   **Primary Color**: `#E91E63` (A bold and distinctive hue for key interactive elements).
-   **Neutral Color**: `#212121` (A deep tone used for backgrounds and subtle contrasts).

### Typography
We use a curated set of fonts to establish a clear hierarchy and enhance readability.
-   **Headline Font**: `manrope` (Used for prominent titles and headings, conveying strength and modernity).
-   **Body Font**: `inter` (Selected for its legibility and versatility in long-form content).
-   **Label Font**: `inter` (Applied to UI labels and smaller text elements for clarity).

### Shape and Form
The geometric properties of our UI elements contribute to the overall aesthetic.
-   **Roundedness**: `2` (Corners have a moderate rounding, balancing approachability with structure).

### Spacing
Our spacing system ensures visual balance and improves content readability.
-   **Spacing**: `2` (A normal level of whitespace, providing adequate breathing room between elements without feeling sparse).
